---

### 📋 Project Overview

Analyzed Zomato restaurant dataset (9,551 records) covering **15 countries** to extract actionable business insights.

---

### 🛠️ Technologies Used

- Python (Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn)
- Exploratory Data Analysis
- Geospatial Analysis (DBSCAN Clustering)
- Restaurant Chain Performance Analysis

---

### 🔥 Key Deliverables

#### Level 1
- Price range distribution & percentage analysis
- Online delivery impact on ratings
- Rating distribution analysis

#### Level 2
- Geospatial clustering of restaurants using **DBSCAN**
- Restaurant chain performance ranking
- High-performing chains identification

---

### 📈 Major Insights

- **25.66%** restaurants offer online delivery
- Restaurants with online delivery have **higher average rating** (3.25 vs 2.47)
- Top chains: `Barbeque Nation`, `AB's Absolute Barbecues`, `Toit`
